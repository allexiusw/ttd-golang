# TDD Golang
Go applications writen line by line using TDD.

### To setup the Development environment
```bash
git clone git@github.com:allexiusw/ttd-golang.git
cd ttd-golang
code .
```

### Installing dependencies
```bash
go go mod init store 
go build -v
```

### Run server
```bash
go task.go
```

### Running the tests
```python
go test -v
```

Happy Hacking :)
